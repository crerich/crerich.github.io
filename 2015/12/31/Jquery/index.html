基础选择器
$('*')                 //选取所有的元素(通配选择器)
$('#divtest')          //获取id名为divtest的元素(id选择器)
$('div')               //获取所有的div元素(标签选择器)
$('.red)               //获取所有的class名为red的元素(类名选择器)
$('ul li')             //获取ul下面的所有的li元素(层级选择器&父子选择器)
$('ul>li')            //获取ul“子节点”中所有的li元素(注意：它比$('ul li')的范围要小)
$('p + div')            //获取p元素后面的第一个兄弟元素div
$('p ~ div')            //获取p元素后面的所有兄弟元素div

过滤性选择器
$('ul li:first')        //选取ul下面的第一个li元素(css3选择器)
$('ul li:last')     //选取ul下面的最后一个li元素(css3选择器)
$("a[href = 'www.baidu.com']")  //选取所有href属性为www.baidu.com的a元素
$('ul li:eq(2)')    //选取ul下面的第三个li元素

    • JQuery操作元素

      关于操作DOM元素    这里主要讲css()、attr()、removeAttr()、addClass()、removeClass()的方法。
说明：css()、attr()具有getter&setter 也就是既可以获取属性值也可以设置属性值
1.css()   //设置元素的样式 (以a标签为例)
    <a href="www.baidu.com" style="background:red">Hello 2016!</a>
    $('a').css('background');  //获取a标签的background属性对应的值即"red"
    $('a').css('background','blue'); //设置a标签的背景色为blue会覆盖之前的red(这是一个样式的写法)
    $('a').css({'background':'blue','font-size':'20px'}) 
    //这是多个样式的写法，注意和一个样式的区别（"{}"、","、“:”）

2.attr和css差不多，只不过一个是对样式进行操作一个是对属性进行操作，注意区分就ok.

3.$('a').removeAttr('href')  //移除a元素的href属性

4.$('a').addClass('haha')    //给a标签加上“haha”的类名

5.$('a').removeClass('haha')  //移除a标签的“haha”类名

    • JQuery DOM操作

      关于DOM操作这里以“<a href="www.baidu.com">Hello 2016!</a>”为例。

1.append(content)         //向指定的元素内追加内容，content可以是字符、html元素标记、或者返回字符串内容的函数
    $('a').append($('<span>Bye 2015</span>'))  
    //结果<a href="www.baidu.com">Hello 2016!<span>Bye 2015</span></a>

2.prepend(content)        //和append相反
    $('a').prepend($('<span>Bye 2015</span>')) 
    //结果<a href="www.baidu.com"><span>Bye 2015</span>Hello 2016!</a>

3.before(content)         //在元素前插入内容
    $('a').before($('<span>Bye 2015</span>'))  
    //结果<span>Bye 2015</span><a href="www.baidu.com">Hello 2016!</a>

4.after(content)            //在元素后加入内容
    $('a').after($('<span>Bye 2015</span>'))  
    //结果<a href="www.baidu.com">Hello 2016!</a><span>Bye 2015</span>

5.clone()                    //生成一个被选元素的副本
    $('a').clone()      //复制a元素

6.wrap(wrapper)                        //进行元素的包裹  包裹元素本身
    $('a').wrap('<div></div>')        //结果<div><a href="www.baidu.com">Hello 2016!</a></div>

7.wrapInner()                            //进行元素的包裹  包裹元素中的内容
    $('a').wrapInner('<strong></strong>')    
    //结果<a href="www.baidu.com"><strong>Hello 2016!</strong></a>

8.remove()                            //删除所选元素本身和子元素

9.empty()                                //删除所选元素的子元素

10.each()                                //遍历指定的元素集合，在遍历时，通过回调函数返回遍历元素的序列号
调用格式为$(selector).each(function(index));

    • JQuery事件处理

      JQuery定义了一个统一事件API来解决IE与所有其他浏览器不用的事件API
JQuery定义了简单的事件注册方法，可用于常用和普适的每一个浏览器事件。如：click、mouseover、mouseout、mousemove、
focus……。

    1.预加载事件      $(function(){})     老式写法：$(document).ready(function(){})

2.click()               //点击事件
    $('p').click(function(){
        $(this).css('background','red');
    })
    //单击任意<p>时，使其背景颜色变为红色

3.hover()             //用来给mouseover()、mouseout()事件注册处理程序和<a>标签的hover效果一样
    $('a').hover(function(){
        $(this).css('background','red');
    })
    //模拟a标签的hover效果

4.bind()                //对元素进行事件的绑定

    $('a').bind(function(){
        mouseover:function(){$(this).css('background','red');},
        mouseout:function(){$(this).css('background','blue');}
    })
    //给<a>标签绑定了mouseover和mouseout事件

5.unbind()            //对元素进行事件的移除

    $('a').unbind('mouseout');
    //继上例移除<a>标签的mouseout事件

6.toggle()            //在元素的click事件中绑定两个或两个以上的函数，同时，它还可以实现元素的隐藏与显示的切换
//格式：$(selector).toggle(fun1(),fun2(),...funN())

$('a').toggle(function(){$(this).html('哈哈')},function(){$(this).html('呵呵')})
//第2n-1次点击<a>标签内容为’哈哈‘,第2n次点击<a>标签内容为’呵呵‘。

7.one()                //可以绑定元素任何有效的事件，但这种绑定的事件只会触发一次。
    一些方法的比较
    1. html()、text()、val()
    $('a').html()            //获取<a>标签的内容 和innerHTML类似
    $('a').text()            //获取<a>内的文本内容（包括子节点）
    $('input').val()        //获取input标签的value属性值

2. width()、innerWidth()、outerWidth()
//width()获取宽度不包括padding值
//innerWidth()获取宽度包括padding值   类似clientWidth
//outerWidth()获取宽度包括padding和margin值   类似offsetWidth

    附上选项卡的代码

    html部分:
    <ul>
        <li class="on">罗氏情怀</li>
        <li>思想聚焦</li>
        <li>预见2016的自己</li>
    </ul>
    <div class="on">
        你可以嘲笑我有幻觉,但我觉得如果他们还活着的话，不会有现在的你。
    </div>
    <div>
        学的不仅是技术，更是梦想。再牛逼的梦想，也抵不住你傻逼似的坚持。
    </div>
    <div>
        我只想安静地努力一会儿。
    </div>    
css部分:
    ul{
            list-style-type: none;
            margin: 0;
            padding: 0;
        }
        li{    
            float: left;
            width: 80px;
            height: 30px;
            text-align: center;
            line-height: 30px;
            border: 4px solid #fff;
            background: #089574;
        }
        li.on{
            background: #ccc;
            border: 4px solid #ccc;
        }

        div{
            font-size: 20px;
            width: 300px;
            height: 200px;
            background: #ccc;
            padding: 30px;
            display: none;
        }
        div.on{
            clear: left;
            display: block;
        }    
js部分:

    $(function(){
        $('ul li').each(function(index){
            $(this).click(function(){

                $('ul li.on').removeClass('on');
                $('div.on').removeClass('on');

                $(this).addClass('on');
                $('div').eq(index).addClass('on');
            })
        })
    })

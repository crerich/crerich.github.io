  • html标签回顾

    html 根标签
    head (title script link style meta系列总结 base(为页面上的所有链接规定默认地址或默认目标))
    body 内容区域
    h1-h6 标题标签 默认的margin
    p 段落标签 默认的margin
    strong 表示更强烈的强调
    a href=”url” title=”” target=”_blank
    img src=”url” title=”鼠标滑过图片时显示的文本” alt=”图像的描述性文本
    q 端文本引用
    blockquote 长文本引用
    br 换行标签
    hr 分割线标签
    address 双标签 为页面加入地址信息 样式为斜体
    code 单行代码
    pre 预格式化的文本 即多行代码

  • html列表标签

    • 无序列表 ul li 组合标签 ul li 间不能加入其他标签
    • 有序列表 ol li 组合标签 ol li 间不能加入其他标签
    • 自定义列表 dl dt dd 组合标签 字典模式

  • html表格标签

    • table
    • tbody
    • tr 表格的一行
    • th 表格表头
    • td 表格的一个单元格
    • caption 表格标题
    • summary 表格摘要

  • html表单标签

    • form action 输入的数据被传送到的地址 method 数据传送的方式 get/post
    • input type = text/password/radio/checkbox/submit/reset/hidden/button… value 默认值 name 文本框命名
    • textarea 文本域 可设置resize属性 cols 多行输入域的列数 rows 多行输入域的行数
    • label 为input元素定义标注 label 的for属性与input的id属性关联 可提高用户体验
    • select 下拉列表框
    • multiple 多选功能
    • option 下拉选项

  • 几点注意事项:

    • Html标签不区分大小写，但是建议小写,闭合标签
    • 标签的默认样式
    • 行级标签、块级标签、其他标签的区别 附图标签类型
    • 标签对SEO的影响 例如a标签需加title的属性等

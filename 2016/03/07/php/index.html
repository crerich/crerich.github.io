  • php历史及特点

1994年php1.0发布 –> 1995年php2.0加入了mysql问世 –> 1997年php3.0 –> php4.0、php5.0 –> 2015年php7.0的震撼来临。
php以其上手快、易修改、发布快捷等特点一统后端，可以说是后段最好的语言没有之一。

  • 旁敲侧听

    • Apache

      Apache是web服务器软件、可以运行在几乎所有的计算机平台上面、最流行的web服务器软件之一

    • MySQL

      关系型数据库管理系统、拥有体积小，速度快，开放源码等优点。

  • 基础语法

    • PHP标记
      <?php ?>
    • PHP注释
      /**/多行注释 //单行注释
    • 变量
      以$开头，后面跟变量名(变量字母、数字、下划线组成，不以数字开头，变量名区分大小写) 例:$age = 18;
    • 变量的赋值
      1.$var_1 = “hello”; //值赋值: 将赋值表达式的值复制给变量(直接赋值)
      2.$var_2 = &$var_1; //引用赋值：一个变量与另一个变量的引用
    • 超全局变量($_SERVER)
      php提供了很多有用预定于的变量如:
      $_SERVER[‘SERVER_NAME’]; : 当前运行脚本所在的服务器的主机名
      $_SERVER[‘REMOTE_ADDR’] : 客户端IP地址
      $_SERVER[‘REQUEST_URI’] : URL的路径部份
      $_SERVER[‘HTTP_USER_AGENT’] : 操作系统和浏览器的有关信息
  • 常量 (内置常量&&魔术常量)
    define(‘PI’,3,1415926)

  • 数据类型
    1.整型
    $age = 18;
    2.浮点型
    $num = 5.39;
    3.布尔型
    $bo = TRUE;
    4.字符串
    $a = “hello”;
    $b = “he{$a}llo”;
    $b = “he’.$a.’llo”; //.是php中字符串拼接符号类似于js中“+”

  • 复合数据类型
    1.数组
    $week = array(“3”,”4”,”2”);
    2.对象
    $db = new db;

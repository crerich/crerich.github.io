正则表达式的模式匹配

正则表达式的小憩

重要知识点：

分组:     reg1 = /['"][^'"]*['"]/;    reg2 = /(['"])[^'"]*\1/;     reg1 = reg2;
    带圆括号的表达式允许同一正则表达式的后部引用前面的子表达式如reg2的表达式(['"])可以用\1来表示。
    (因为子表达式可以嵌套另一个子表达式，所以它的位置是参与计数的左括号的位置。例如:/([Jj]ava([Ss]cript)?)\sis\s(fun\w*)/ 此式的\2就是([Ss]cript))
    若创建一个不带数字编码的引用，也可以对子表达式进行分组。但此时应用"(?:"和")"。例如:/([Jj]ava(?:[Ss]cript)?)\sis\s(fun\w*)/ 此式的\2就是(fun\w*)。

String支持的4种正则表达式的方法search()、replace()、match()、split()。
    一.search():返回与之匹配的子串的起始位置。若无，返回-1。若search()的参数不是正则表达式，则通过RegExp构造函数将其转换成正则表达式。
           search方法不支持全局检索，因为它会忽略正则表达式参数中的修饰符g(即:全局匹配)
    二.replace():用以执行检索与替换操作。若replace()的第一个参数不是正则表达式，将直接搜索这个字符串。若第一个参数是正则表达式并带有g的话，
则替换所有与正则表达式匹配的元素。否则只替换第一个。
    三.match():返回一个由匹配结果组成的数组。 它的唯一参数是一个正则表达式或者通过RegExp()构造函数将其转换成正则表达式。
    四.split():这个方法用以将调用它的字符串拆分成一个子串组成的数组 它的参数可以是一个正则表达式。
RegExp对象的属性：source、global、ignoreCase、multiline、lastIndex
    source:只读的字符串，包含正则表达式的文本。
    global:只读的布尔值，说明这个正则表达式是否带有修饰符g。
    ignoreCase:只读的布尔值，说明这个正则表达式是否带有修饰符i。
    multiline:只读的布尔值，说明这个正则表达式是否带有修饰符m。
    lastIndex:可读/写的整数。如果匹配模式带有g修饰符，这个属性存储在整个字符串中下一次检索的开始位置。
RegExp对象的exec()、test()方法。
    exec():对一个指定的字符串执行正则表达式。
    test():如果有匹配正则表达式的字符返回true否则返回false。

数组方法

数组常用方法：

1.Array.prototype.join() 数组的指定格式的分割

var arr = [1,2,3];
console.log(arr.join());  //默认情况下为逗号隔开    
console.log(arr.join('-')); //添加字符使之隔开

function repeat(str,n){  //利用'a'做为分隔符来repeat 'a'
    var arr = new Array(n+1);
    return arr.join(str);
}
console.log(repeat('a',4)); //aaaa
